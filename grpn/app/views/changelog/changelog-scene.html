Release Notes
grpn 1.2.0 (1/9/2010)
  • Feature: Added 'Share' option to the deal details screen. This button is located in the bottom left corner of the screen, by tapping it you will be able to share the Groupon via email or text message. Sharing via other applications such as twitter/facebook are in the works.
  • Enhancement: The 'Discussion' button has been made smaller and moved into the lower right corner of the deal details screen. This should make the overall screen feel less cluttered.
  • Bug Fix: The text describing number of Groupons sold for a given deal now properly reflects if more need to be bought to get the deal, if the deal is on, or if the deal is sold out.
  • Bug Fix: Date processing has been enhanced with extDate.js library.
grpn 1.1.0 (released 1/1/2011)
  • Feature: Added 'Discussion' option to the deal details screen. Tapping this button allows you to view user generated discussion regarding the deal.
  • Feature: Added 'Groupon Says...' feature. This screen allows you to randomly read some of the tidbits of information Groupon provides regarding a variety of subjects. Accessible via the application menu.
  • Feature: Added 'Release Notes' menu item to help users track application changes. Accessible via the applications menu.
  • Enhancement: Added the number of Groupons sold for each deal, placed next to the "Discussion" button.
  • Enhancement: Toggling favorites now gives some extra context about how to 'unfavorite' the item
  • Bug Fix: 'View Map' button should now work correctly.
